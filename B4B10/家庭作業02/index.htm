SOUND VOLTEX VIVID WAVE-段位認定模式(SKILL ANALYZER)

此頁面僅介紹SDVX其中一項遊戲模式，更多介紹可參閱官網或維基百科，或自行前往機聽遊玩^0^

段位認定模式是什麼？

段位認定模式，以遊玩規定的歌曲測試自己實力（SKILL LEVEL）的模式。連續通過對應的樂曲即算作通過（「分析成功」），獲得相應的SKILL LEVEL和等級。

SKILL LEVEL的名字及顏色如下。

  • Lv.01 岳翔（銅）
  • Lv.02 流星（藍）
  • Lv.03 月衝（黃）
  • Lv.04 瞬光（白）
  • Lv.05 天極（水藍)
  • Lv.06 烈風（翡翠綠）
  • Lv.07 雷電（紅）
  • Lv.08 麗華（粉紅）
  • Lv.09 魔騎士（銀）
  • Lv.10 剛力羅（金）
  • Lv.11 或帝滅斗（深紅）
  • Lv.∞ 暴龍天（紫背景金字）

遊玩最近一次所獲得最高 SKILL LEVEL 所追加的課題，並且挑戰分析成功可以獲得銀色邊框，遊玩所獲得最高 SKILL LEVEL 的所有課題，並且挑戰分析成功可以獲得金色邊框。
將所獲得最高 SKILL LEVEL 之前的所有課題全部達成相應的達成率後可以獲得後光邊框。

如何選擇段位認定模式

    1. 投30元、刷紀錄卡
    2. "選擇SKILL ANALYZER"
    3. 選好角色後(非必要)選擇"GAME START"
    4. 進入選擇段位及課題畫面
    5. 根據自身能力及喜好選擇欲考取段位及課題組曲

過關條件

在血條歸零前，通關三首課題指定曲，即可拿到該段位
  • 在遊玩課題曲過程中，若失誤將會扣血
  • 血條歸零將會強制結束遊戲，也無法取得該段位
